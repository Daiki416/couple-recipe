# 要件

## 目的

夫婦で使うレシピ管理アプリ。

- レシピを保存・検索・共有する
- スマホ利用を前提とする
- 将来的にAIでレシピ提案を行う

## MVP

### 認証
- Supabase Auth

### 世帯
- 1ユーザーは1世帯
- 世帯内でレシピを共有
- 招待コードで参加

### レシピ
- CRUD
- 材料
- 手順
- 写真
- メモ
- 出典URL
- 人数
- 評価
- お気に入り
- 調理回数
- 最終調理日

### 検索
- レシピ名
- 食材名
- お気に入り
- 評価
- タグ

## 非機能

- RLS必須
- 無料枠運用
- モバイル優先
